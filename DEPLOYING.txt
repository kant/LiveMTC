*** Deploying is only done by Showsync **

Prepping:

- Have the Showsync Developer ID certificate installed.

- Make sure you have xcode 11.3.1 or later (download from developer account if necessary:https://developer.apple.com/download/more/, search for xcode 11.3.1)

- Ask Hidde or Mattijs for the showsync developer id password

- Do this once to store the password on your machine:
xcrun altool --store-password-in-keychain-item "SHOWSYNC_AC_PASSWORD" -u "build@showsync.com" -p <showsync-developer-id-password>

- Install this once: brew install akeru-inc/tap/xcnotary


Every release:
- Build the application from Max (to a folder App

- add a CFBundleName field to the Contents/Info.plist (right-click, Show Package Contents)

- codesign -s "Developer ID Application: Stichting SMG (P92XQLN5T4)" -v -f --deep --options runtime <path-to-app>/Live\ MTC\ Bridge.app

- xcnotary notarize <path-to-app>/Live\ MTC\ Bridge.app  --developer-account build@showsync.com --developer-password-keychain-item SHOWSYNC_AC_PASSWORD




Background info:
